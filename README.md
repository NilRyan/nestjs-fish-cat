
## Description
A GraphQL Monolithic API for FishCat Dating App - Built with NestJS, TypeScript, PostgreSQL

## Features
  -  Login / Registration
  -  Matching
  -  Real Time Chat
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker-compose up
$ npm run start

# watch mode
$ docker-compose up
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
